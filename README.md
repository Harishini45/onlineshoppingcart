# onlineshoppingcart
Online Shopping Cart Project This project is an online shopping cart application developed using Java, SQL, Servlet, JSP, HTML, CSS, Bootstrap, and JavaScript. It provides users with a platform to browse, select, and purchase products online, simulating the experience of an e-commerce website.

Features

1. Product Catalog

   he application offers a comprehensive catalog of products, categorized for easy navigation.
Each product is presented with details like name, price, description, and an image.

2. Shopping Cart
Users can add products to their shopping cart with a single click.
The cart dynamically updates to display the total number of items and the total cost.
Users have the ability to view and edit the contents of their cart, including updating quantities and removing items.

3. User Authentication
   
The application supports user authentication to ensure secure access to user-specific features.
Users can create an account, log in, and log out.
Only authenticated users can proceed to checkout.

4. Checkout Process
   
Authenticated users can proceed through the checkout process to finalize their purchases.
The checkout process includes entering shipping and payment information.
Once the order is confirmed, users receive a confirmation email with the details of their purchase.

5. Admin Panel
The application includes an admin panel accessible only to authorized administrators.
Admins can add, edit, or delete products from the catalog.
They can also view and manage orders placed by users.

Technologies Used

Backend: Java, Servlet
Frontend: HTML, CSS, Bootstrap, JavaScript
Database: SQL (e.g., MySQL)
Authentication: Sessions or JWT (JSON Web Tokens)
